MtM+ Toolbox

    A proactive problem-solving solution for MtM+. I designed and developed this all-in-one app to help sales win customers' trust.

OVERVIEW

As a fast-paced IoT startup, MtM+ develops many different solutions (sensor + mobile app) in a short time. However, sales have difficulty while demoing because some solutions are using similar app interface but with different algorithms. I observed the situation and initiated this Toolbox solution, an all-in-one app, to solve the sales team's problems. By improving the demo quality, we built trust with customers and increased the company revenue.

TYPE

Initiative project

TIMELINE

4 month (Mar-Jun 2018)

TOOLS

Sketch, Android Studio

CONTRIBUTION

Research, Design, Development

OBSERVATION

IDEATION

Besides observation, I talked to the sales team to gain more understanding for their pain points. I wrote down all their complaints and tried to find out all the hidden problems. With the problems in mind, I listed all the solutions that had come out in my mind.

DESIGN

Visualized the idea and hold user ability test to make sure the user flow and information are followed the user mental model.
Goal 1 - Simplify the user flow and combining all demo apps into one.
By showing the products on the main page, sales now can choose different products they want to demo directly in one app. Also, I reduce 3 clicks to 1 click by using code to auto-connect the sensor.
Goal 2 - Improving the design to make it more productive
I defined the company's design system and unify the design style.
After designing is done, I started to code the application and to make all functions works. After a few times of testing the application is now can be downloaded on Google Play.
